Team plugin is fully Responsive WordPress Team member display plugin to create & manage your Team page easily. Your can display your Team members profile in views.

== Description ==
**CA Team**
CA Team plugin is fully responsive & mobile friendly team members display plugin in WordPress. You can display your team members in different layouts.
CA Team allow you to create team member profiles. You can select layout for profile image square or rounded. So you can arrange your team member profile easily using square or rounded profile image.
Plugin have different option you can easily control which layout you want ,color, Socail media buttons enable or disable option, detail page link option etc.
Team Plugin shortcode is included in settings menu.
== Features ==
* Fully Responsive & Mobile Friendly
* 2 Different layouts
* Custom meta filed
* Color control
* Many other options
= Available fields for team member =
* Title/Name (Post Title)
* Description (Custom field)
* About (Custom field)
* Position (Custom field)
* Email (Custom field)
* Skills (Custom field)
* Social links (FB, Twitter, LinkedIn, Youtube)
== Installation ==
==Automatic installation== 
Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of this plugin, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.
In the search field type \"Centangle-Team\"  and click Search Plugins. Once you've found this plugin, you can install it by simply clicking Install Now

==MANUAL INSTALLATION==
Download and unzip the plugin
Upload the entire plugin/ directory to the /wp-content/plugins/ directory
Activate the plugin through the Plugins menu in WordPress
